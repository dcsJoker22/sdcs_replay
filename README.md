# SDCS Campaign Replay Viewer

Interactive web-based replay of [Strategic DCS](https://strategic-dcs.com) campaign sessions. Displays aircraft tracks, ground units, base ownership, kill events, objective progress, and pilot telemetry on a live map.

## Live Demo

Hosted at [strategic-dcs.com](https://strategic-dcs.com) — currently in development.

---

## Stack

| Component | Technology |
|-----------|-----------|
| Parser | Python 3 — converts Tacview `.acmi` files to session JSON |
| Viewer | Single-file HTML + Leaflet.js — no build step |
| Data | Pre-processed JSON files, one per campaign session |
| Hosting | Static file server (any) |

## Setup

### Requirements

```bash
pip install watchdog   # for watch_acmi.py only
```

### 1. Parse a campaign folder

```powershell
# Parse all ACMIs in a campaign folder (output goes to raw/<campaign>/data/ automatically)
Get-ChildItem -Path raw -Recurse -Filter *.acmi | ForEach-Object {
    python parse_acmi.py $_.FullName
}
```

### 2. Build the campaign index

```powershell
python build_campaigns.py
```

Scans all `raw/*/data/session_*.json` and writes `public/campaigns.json`.
Campaign names and theatre IDs are auto-detected from folder names.

### 3. Serve locally

```powershell
python -m http.server 8080 --directory public/
```

Open `http://localhost:8080/sdcs_replay.html`.

## Adding a New Campaign

1. Create `raw/YYYY-MM-DD CampaignName/` (date = campaign start, include map name)
2. Drop ACMI files in
3. Run `parse_acmi.py` on them
4. Run `build_campaigns.py`

Folder name auto-detection:
- Contains `Caucasus` → Caucasus (id 190)
- Contains `Germany` → Germany (id 192)
- Contains `Syria` → Syria (id 193)
- Contains `Gulf` or `Persian` → Persian Gulf (id 189)

## Data Pipeline

```
.acmi → parse_acmi.py → session.json → build_campaigns.py → campaigns.json → viewer
```

## Viewer Features

- Campaign selector (newest first) + session selector (oldest first, newest pre-selected)
- Player/AI aircraft icons — 9 type-specific SVG silhouettes
- Heading rotation, smooth interpolation, altitude/Mach telemetry
- Air-launched weapon tracks (missiles, bombs, rockets — AAA/tank shells filtered)
- Kill events, base lifecycle, factory capture, objective progress bar
- Ground unit diamonds, segmented fading trails
- Player detail panel, kill feed with PvP highlighting
- Filters: player aircraft, AI, ground, trails, kills, bases, weapons, FLOT
- Map auto-centres to correct theatre on campaign switch
